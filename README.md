# OpenWeather API Project

## Overview
This project is a web application that uses the **OpenWeather API** to fetch real-time weather information for any city. It is built using **HTML, CSS, and JavaScript**, demonstrating API requests, error handling, and dynamic content rendering.

---

## Features
- **Search Weather by City Name**  
- **Display Weather Details**: temperature, humidity, wind speed, and weather description  
- **Error Handling** for invalid city names or failed requests  
- **Dynamic Updates** without page reload  
- **Responsive Design** for mobile and desktop  

---

## Installation

1. Clone the repository:
```bash
git clone https://github.com/yourusername/openweather-api.git

```
2. Navigate to the project folder:

cd openweather-api

3. Open index.html in your browser.

4. Create a .env file (optional) to store your API key securely:

API_KEY=your_openweather_api_key
